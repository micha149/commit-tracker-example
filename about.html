About CommitTracker

CommitTracker is a simple task management application developed as part of the Git workshop "Saubere Commits f√ºr eine nachhaltige Code-Historie".

Purpose

This application serves as a practical example for demonstrating various Git techniques and best practices for organizing commits. It provides a realistic codebase that workshop participants can use to practice different Git operations.

Features

  • Create and manage tasks with titles, descriptions, categories, and priorities
  • Set due dates for tasks
  • View task statistics and analytics
  • Data persistence using localStorage

Technology

CommitTracker is built using basic web technologies to ensure accessibility for all developers:

  • HTML5 for structure
  • CSS3 for styling
  • JavaScript (ES6+) for functionality
  • Vite as a build tool
  • localStorage for data persistence
